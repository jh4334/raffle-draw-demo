# 공식 번호 추첨 시스템

공공기관의 엄정한 추첨 자리에서 사용할 수 있도록 만든 모바일 대응 웹 추첨 프로그램입니다. 기본 설정은 **1-30번 중 당첨 번호 추첨**입니다.

## 바로 테스트

GitHub Pages:

https://jh4334.github.io/raffle-draw-demo/

## 주요 기능

- 기본 추첨 범위: `1-30`
- 당첨 수 지정
- 중복 없는 무작위 번호 추첨
- “추첨 대기 → 추첨 진행 중 → 결과 확정” 단계 표시
- 기관명/행사명/추첨 범위/당첨 수 조건 요약
- 재추첨 시 이전 결과를 감사 로그에 보존
- 공식 결과 문구 복사
- 모바일·태블릿·노트북 대응 UI
- 공공기관 분위기에 맞춘 IBM Carbon 스타일 UI

## 현장 사용 순서

1. 기관명과 행사명을 입력합니다.
2. 추첨 범위가 `1-30`인지 확인합니다.
3. 당첨 수를 입력합니다.
4. 입회자에게 “진행 전 낭독용 요약”을 확인시킵니다.
5. **추첨 시작 및 결과 확정**을 누릅니다.
6. 필요하면 **결과 복사**로 공식 결과 문구를 복사합니다.
7. 재추첨이 필요하면 **재추첨**을 누릅니다. 이전 결과는 감사 로그에 남습니다.
