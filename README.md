# 공식 번호 추첨 콘솔

공공기관의 엄정한 추첨 자리에서 모바일 또는 태블릿으로 바로 진행할 수 있도록 만든 정적 웹 추첨 콘솔입니다. 기본 설정은 **1-30번 중 당첨 번호 추첨**입니다.

## 바로 테스트

GitHub Pages:

https://jh4334.github.io/raffle-draw-demo/

## 주요 기능

- 기본 추첨 범위: `1-30`
- 당첨 수 지정
- 중복 없는 무작위 번호 추첨
- “추첨 대기 → 추첨 진행 중 → 결과 확정” 단계 표시
- 기관명/행사명/추첨 범위/당첨 수 조건 요약
- 재추첨 시 이전 결과를 감사 로그에 보존
- 공식 결과 문구 복사
- 모바일·태블릿·노트북 대응 UI
- 모바일 우선 프리미엄 공공기관 콘솔 UI

## 현장 사용 순서

1. 상단에서 기관명과 행사명을 확인합니다.
2. 기본 조건 그대로 진행하면 **공식 추첨 시작**을 누릅니다.
3. 조건 변경이 필요하면 **현장 조건 확인 및 변경**을 열어 기관명, 행사명, 번호 범위, 당첨 수를 조정합니다.
4. 결과 번호가 확정되면 감사 로그에 회차별 기록이 자동 보존됩니다.
5. 필요하면 **결과 문구 복사**로 공식 결과 문구를 복사합니다.
6. 재추첨이 필요하면 **재추첨 진행**을 누릅니다. 이전 결과는 감사 로그에 남습니다.
